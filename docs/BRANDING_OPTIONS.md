# 🎨 Identité Visuelle & Branding

Choix et direction artistique pour le projet.

## 1. Options d'Identité

### 👻 Option A : GhostPoll (Privacy & Fun)
- **Concept** : Le vote fantôme, là un instant, disparu ensuite.
- **Mascotte** : **Spooky**, un petit fantôme pixel-art ou vectoriel avec des lunettes noires.
- **Palette** :
  - Primary: `#8B5CF6` (Violet Neon)
  - Accent: `#10B981` (Vert Slime/Success)
  - Background: `#09090b` (Rich Black)
- **Ton** : Décalé, complice, mystérieux.

### ⚡ Option B : FlashVote (Speed & Tech)
- **Concept** : Ultra-rapide, instantanéité.
- **Mascotte** : **Spark**, un éclair stylisé.
- **Palette** :
  - Primary: `#3B82F6` (Electric Blue)
  - Accent: `#F59E0B` (High Voltage Yellow)
  - Background: `#0F172A` (Navy Blue)
- **Ton** : Efficace, énergique, direct.

### 🔥 Option C : RawVote (Warmth & Truth)
- **Concept** : Feedback à chaud, authenticité.
- **Mascotte** : **Pyro**, une petite flamme.
- **Palette** :
  - Primary: `#F97316` (Hot Orange)
  - Accent: `#EF4444` (Passion Red)
  - Background: `#18181B` (Zinc Dark)
- **Ton** : Humain, chaleureux, honnête.
